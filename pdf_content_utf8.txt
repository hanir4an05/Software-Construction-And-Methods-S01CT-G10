Project Title: Student Learning Management System (SLMS) 
What CSEB5223 lab is about?  
The lab activities let you practice your coding skills. You'll use what you learned in class to 
solve a real problem. We want you to build good habits and follow coding rules. 
How we'll do it: 
1. Step by step: We'll break the project into smaller parts. You'll work on one part at a 
time. This makes it easier to learn. 
2. Build bit by bit: Try to have something working after each lab. Even if it's small, it's 
progress. This helps you see how you're improving. 
3. Keep it tidy: We'll use the same rules to write code. This includes how we name 
things and add comments. Clean code is easier to read and fix later. 
By doing this, you'll get better at making software. You'll also learn good habits for working 
as a real programmer. 
Let's start coding and have fun! 
Main reference of learning management system to be built: Moodle, codebase.
