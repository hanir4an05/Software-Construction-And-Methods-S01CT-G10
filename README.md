 Student Learning Management System (SLMS)

The Student Learning Management System (SLMS) is a web-based platform designed to simplify academic management for students, teachers, and administrators. It provides a centralized environment for course, student and enrollment Management.

- Features

-Course Management System Features

1. Add the course to the system.
2. Search Course by Course Code
3. Edit Course content by entering course code
4. Delete the Course by entering Course code
5. Display all courses 

-Student Management System Features

1. Add the student to the system.
2. Search Student info by entering Student ID
3. Edit Student Info by entering Student ID
4. Delete Student ID by entering Student ID
5. Display all student registered

- Enrollment Management System Features

1.Enroll registered student to the course
2.Find Courses by entering student ID
3.Find Student  by entering Course ID
